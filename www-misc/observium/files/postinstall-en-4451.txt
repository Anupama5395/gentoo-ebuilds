
  1. Create a new MySQL database:
  mysqladmin create observium

  2. Connect to this database and create a user:
  mysql -h localhost -u root -p observium
  mysql> GRANT ALL PRIVILEGES ON observium.* TO 'observium'@'localhost'
      -> IDENTIFIED BY '<observium db password>';

  3. Edit ${VHOST_ROOT}/htdocs/${PN}/config.php and set database settings.

  4. Add an admin user, with access-level 10:
  php ${VHOST_ROOT}/htdocs/${PN}/adduser.php admin <password> 10

  5. Add an SNMP host to be monitored:
  php ${VHOST_ROOT}/htdocs/${PN}/add_device.php <hostname> <community> v2c

  6. Perform initial data-acquisition:
  php ${VHOST_ROOT}/htdocs/${PN}/discovery.php -h all
  php ${VHOST_ROOT}/htdocs/${PN}/poller.php -h all

  7. Login to http://observium.${VHOST_HOSTNAME/localhost/$( hostname -f )}/
